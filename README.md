# 🔥 Mathstick

**Mathstick** is a free web-based matchstick puzzle game built by Derek Liu, creator of Trangram, for all puzzle lovers.

The latest version comes with a **powerful Puzzle Maker**, letting players not only solve challenging puzzles but also design and share their own.

All illustration assets in this game are crafted using [Trangram](https://www.trangram.com/).

---

## 🚀 Features

- 🧠 **Procedurally generated puzzles**  
  Automatically generated puzzles with guaranteed solutions and increasing difficulty.

- 🛠️ **Powerful Puzzle Maker**  
  Create your own matchstick puzzles with different stick styles.

- 🔗 **Easy sharing**  
  Share puzzles via link or social media to challenge others.

- 🎨 **Clean, responsive UI**  
  Smooth experience across desktop and mobile devices.

- ⚡ **Lightweight & fast**  
  Built as a fast, pure web application.
